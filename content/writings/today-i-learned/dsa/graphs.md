## DFS

| time              | O(n)         |
| ----------------- | ------------ |
| When Recursive    | **Set**      |
| Without Recursion | Stack needed |

## BFS

> Usually, non recursive.

| time           | O(n)              |
| -------------- | ----------------- |
| Data Structure | **Queue and Set** |

---

## Dijkstra's Shortest Path

| time           | O(ElogV)                             |
| -------------- | ------------------------------------ |
| Data Structure | **Heap** _(priority queue)_ and Sets |

> E:Edges, V:Vertices
